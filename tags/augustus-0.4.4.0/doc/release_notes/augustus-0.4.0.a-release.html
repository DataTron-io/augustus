2. Point release additions


None

This release allows Augustus to consume and score PMML 4 complaint models. It also address Missing Value Replacement. The substantial changes are in pmmllib:

  • augustus/pmmllib/pmmlBaseline.py
  • augustus/pmmllib/pmmlReader.py
  • augustus/pmmllib/pmmlModels.py
  • augustus/pmmllib/pmmlNaiveBayes.py
  • augustus/pmmllib/pmmlConsumer.py
  • augustus/pmmllib/pmmlElements.py
  • augustus/pmmllib/pmmlModelElements.py
  • augustus/pmmllib/pmmlTree.py
  • augustus/pmmllib/pmmlRegression.py
More information will be made available in a later release.
