2. Point release additions


Tree Model

Augustus now supports the Tree model. All types of trees can be consumed and classification trees can be produced.


Regression Model

Augustus now supports the Regression model.


Unified Consumer

Previously, each type of model had its own consumer script. Now that we've added support for two more models, we've discovered that we only need to have a single consumer script that can consume all types of models. Please replace any uses of AugustusBaselineConsumer or AugustusNaiveBayesConsumer with the new AugustusPMMLConsumer.

3. Bugs fixed


Naive Bayes Improvements

Various minor bugs releated to the Naive Bayes model were fixed.


PMML Builtin Functions

Various builtin functions were clarified in PMML version 3.2 and we've updated our implementation of these to match their description in the standard.
