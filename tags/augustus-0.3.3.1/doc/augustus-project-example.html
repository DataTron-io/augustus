2 Setup

2.1 Copy files

Copy the files from augustus/examples/projectExample into a new directory not in the augustus directory tree. This new directory is the project directory.

For example:

cd ~/augustus/examples
cp -pr projectExample/ ~/project

2.2 Set environment variables

Set the PROJECT environment variable to point to the project directory. Set the PYTHONPATH to point to project and augustus.

For example:

export PROJECT=~/project
export PYTHONPATH=$PYTHONPATH:$PROJECT:~/augustus

2.3 Data

Download one of the traffic-24h*.gz data files, for example traffic-24h-2007-11-01.gz, from the Pantheon Testbed provided by the TeraFlow Testbed and place it in project/data.

Uncompress the data file:

gunzip project/data/traffic-24h*.gz

3 Execution

3.1 Analyses

Choose one of the following:

Occupancy Example - Move to project/occupancyExample/scripts.
Speed Example - Move to project/speedExample/scripts.
Volume Example - Move to project/volumeExample/scripts.

3.2 Create a PMML file

Enter

python produce.py -f traffic-24h* -t 10

to produce a PMML file while outputting timing information every 10% of the way through the data.

3.3 Produce scores from a data file

Enter

python consume.py -b traffic-24h* -f traffic-24h*

to score the same data you used to create the PMML file.

After the producer and consumer complete processing output data will be in the producer and consumer directories.

3.4 Help

If you wish to see help for these programs enter

python produce.py -h
python consume.py -h.
