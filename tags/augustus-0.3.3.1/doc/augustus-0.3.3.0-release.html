2. Point release additions


Performance Improvements

A few methods were optimized after reviewing profiling information from Augustus runs against large datasets.


Fix Width input without new lines.

Fix-width input data can now be read when it does not have End of line characters. Fix-width input files were added in the previous release and this is a feature enhancement.

3. Bugs fixed


Problem scoring non-String data

In tree and Naive-bayes Producers, scoring a non-String field caused erroneous behavior. This has been fixed.


Unnecessary print statements

Extra print statments were removed from the Producer.


Local Transformations

When local transforms are present, Regressions need to update itself (the segment) and the local transformations. This was not being handled correctly in version 0.3.2.
