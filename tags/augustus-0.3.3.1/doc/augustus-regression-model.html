2 Producer

2.1 Description

The producer creates PMML files. It requires a simple PMML file as input upon which it bases the output file. It also requires a configuration file as input to tell it exactly how to make the output file.

2.2 Input PMML File

An example is provided for the reader.

The data dictionary section tells which fields are required as input. The regression model section describes the model. The mining schema tells which fields are required for the actual scoring process or for output. The critical portion of this section is the set of tables specifying values for input fields and the output field.

2.3 Configuration File

An example is provided for the reader.

The configuration file specifies how to score the model. The batch element should be specified if you wish the scoring engine to produce all scores after all the events have been processed. The model element tells where to find the input file and where to produce the output file. The build element identifies the file to be used to be a basis for the resulting model and whether the file contains XML or UniTable formatted data. The segmentation tag allows the user to utilize different tests for different input data. In the example, the user is specifying that they want to use the model solely to score data from the month of December.

2.4 Output PMML File

An example is provided for the reader. Note that the comments from the input PMML and the configuration file were completely removed from the example before the output PMML file was created. The ability to transfer these comments may be added in a later version of Augustus.

This PMML file specifies everything about a valid model. The data dictionary section tells which fields are required either as input or eventual output. The regression model section describes the model. The mining schema tells which fields are required for the actual scoring process. And each Regression section describes a different set of parameters used to score data when the corresponding input conditions (segment) are correct.

2.5 Usage

Assuming that the environment variables are specified correctly (as given in the installation guide and in the FAQ toward the end of this document) the following command will run the producer:

AugustusRegressionProducer -c config -t timing

Where config is the configuration file. If timing is present, the producer will output timing information when every timing% of the data is processed.
