3. Bugs fixed


Issue

This release addresses a change in behavior introduced with the previous version 0.3.3.1. While not necessarily a bug, the new behavior is different enough that it can cause unexpected results with certain types of models. The change introduced is: In batch scoring, results are omitted for segments with no data.

If you are not using segments and batch scoring, then this change did not impact your results.

This release gets rid of this behavior so that models acting with segments can be used without change.
