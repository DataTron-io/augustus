Installing Augustus on Linux

Introduction

This document details the installation of Augustus on Linux. The following instructions are targeted at the Ubuntu Linux distribution, but should be applicable to other distributions with minor changes.

Install Dependencies

Install Python2.5 and Python2.5-dev packages:

Augustus requires Python 2.5. If 2.5 isn't currently installed on your system, it can be installed from the distribution or though apt-get. To install through apt-get:
    • sudo apt-get install python2.5
    • sudo apt-get install python2.5-dev
To install from the distribution:
    • Download the current version of Python 2.5 from:  http://python.org/download/
    • Unpack and install the distribution:
      • tar -zxvf Python-<version>.tgz
      • cd Python-<version>
      • ./configure
      • make
      • sudo make install

Ensure Python symbolic link points to Python 2.5:

    • cd /usr/bin
    • sudo rm python
    • sudo ln -s python2.5 python

Download and Install Numpy:

    • Download the current version of Numpy from http://sourceforge.net/project/showfiles.php?group_id=1369&package_id=175103
    • Unpack the distribution:  
      • tar -zxvf numpy-<version>.tar.gz 
    • Install Numpy
      • cd numpy-<version>
      • sudo python setup.py install
    • Alternatively, use apt-get to install numpy:
      • sudo apt-get install python-numpy

Install Augustus

  • Download Augustus from SourceForge: http://sourceforge.net/projects/augustus
  • Unpack and install the distribution:
    • tar -zxvf augustus-0.3.3.1.tar.gz
    • cd augustus-0.3.3.1
    • sudo python setup.py install

Test Installation

The instructions for running the project examples can be used to ensure that Augustus is installed properly.
